# Rolling Shutter Price Calculator - Project Summary

## Professional Portfolio Entry

**Project Title:** Rolling Shutter Price Calculator - Professional Quotation System

**Overview (100 words):**

Developed a production-ready Rolling Shutter Price Calculator featuring an elegant React-Bootstrap interface with real-time pricing calculations. The application enables businesses to generate professional quotations with automatic area calculations, flexible discount handling, and VAT configuration. Built with React 19, Node.js, and MySQL, the system includes comprehensive quotation management with PDF export capabilities and a secure admin panel for price updates. Implemented role-based authentication, end-to-end type safety with tRPC, and responsive design for desktop and mobile devices. The solution streamlines the quotation process, reducing calculation errors and improving customer service efficiency through automated workflows and professional documentation generation.
